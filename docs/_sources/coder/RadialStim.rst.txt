-------------------------------
Radial Stim
-------------------------------

To import RadialStim, you can either use:
```python
from psychopy_visionscience.radial import RadialStim
```
or, any time after `psychopy.plugins.activatePlugins` has been called:
```python
from psychopy.visual import RadialStim
```
