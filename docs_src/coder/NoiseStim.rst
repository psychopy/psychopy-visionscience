-------------------------------
Noise Stim
-------------------------------

To import NoiseStim, you can either use:
```python
from psychopy_visionscience.noise import NoiseStim
```
or, any time after `psychopy.plugins.activatePlugins` has been called:
```python
from psychopy.visual import NoiseStim
```
